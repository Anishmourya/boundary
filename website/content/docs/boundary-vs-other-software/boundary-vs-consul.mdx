Consul is a service mesh
zero trust, identity-based model
designed for non-human use/configuration
machine-to-machine access
codifies authentication and traffic rules with encrypted traffic, automates identity-based access
discover services, automate network configurations, enable secure connectivity across any cloud or runtime


Boundary human interactive design
human-to-machine access
fine-grained authorizations to distributed networks/systems
