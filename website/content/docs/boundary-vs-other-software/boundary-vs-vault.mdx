Vault is a secrets management software that provides encryption and revocation.

While Vault is capable of managing credentials, its intended scope is in secret management and not session management. Boundary is designed as a stateless system for session management. It can utilize a relational database and efficiently handle hundreds of thousands of ongoing sessions at a time.
